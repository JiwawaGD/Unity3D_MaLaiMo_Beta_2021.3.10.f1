Universal Render Pipeline upgrade
---------------------------------

Supported version: 7.1.8 and up only!

1) Import Old Brick House pack to your URP project, and auto-upgrade all project materials to pipeline! (Edit/Render pipeline menu)
2) Double click on OBH_HDRP_URP upgrade unityasset package in this folder.
3) Enjoy!

High Definition Render Pipeline upgrade
---------------------------------------

Supported version: 7.1.8 and up only!

1) Import Old Brick House to your configured HDRP project!
2) Double click on OBH_HDRP_Upgrade to overwrite project to HDRP!
3) Enjoy!
